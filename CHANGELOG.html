v2.0.1 2026-04-09 Latest

✓ Fixed

  • ✓ PRs with more approvals than changes requested are now correctly shown as approved (green hatching)
v2.0.0 2026-04-08

+ Added

  • + Issue Finder page — find mature open-source issues to contribute to
  • + Issue scoring algorithm based on maturity (labels, maintainer validation, activity, description)
  • + Auto-discovery of repos from your merged PRs
  • + Manual repo management with localStorage persistence
  • + Personal notes on issues stored in localStorage
  • + Loading overlay with progress phases and animated spinner
  • + Stale notes cleanup (removes notes on closed issues or issues with linked PRs)
  • + Per-issue refresh button
  • + Navigation links between PR Dashboard and Issue Finder
  • + Changelog page

~ Changed

  • ~ Extracted all inline CSS/JS from index.html into separate files
  • ~ Reorganized project into assets/css and assets/js directory structure
  • ~ Shared styles and logic (modals, i18n, auth, API) mutualized in common files
  • ~ Translations moved to dedicated i18n/ files
v1.5.0 2026-04-07

  • + Contribution badges column with medal tiers (Bronze to Legendary)
v1.4.0 2026-04-06

+ Added

  • + Celebration feature — confetti and modal when PRs get approved or merged

  • + Review status tracking with "My turn" / "Their turn" badges
  • + Cache system (4h TTL) with manual refresh
  • + Repository filter and search
  • + Old PRs filter (> 2 years)
  • + 5-language support (EN, FR, DE, ES, IT)
  • + 100% client-side — token stored in localStorage only
