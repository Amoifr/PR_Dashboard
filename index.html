How to use this dashboard

Your Pull Requests

This dashboard displays all your PRs (open, merged, closed) across all GitHub repositories where you contribute.

Review Status

Each PR shows its review status: pending, approved, or changes requested. The "My turn" badge indicates when you need to respond.

Cache System

Data is cached for 4 hours to speed up loading. Use the "Refresh" button to force a fresh load from GitHub.

Old PRs Filter

By default, PRs older than 2 years are hidden. Enable the toggle to show them.

Filters & Search

Filter by repository using the buttons, or search by title, author, or label.

GitHub Configuration

100% local and secure
Your credentials are stored only in YOUR browser's localStorage. No data is sent to any third-party server. This page communicates directly with the GitHub API.

How to generate a token?

1. Go to GitHub Settings > Tokens (Fine-grained)

2. Click on "Generate new token"

3. Configure the token:

  • Token name: e.g.: "PR Dashboard"
  • Expiration: according to your preference
  • Repository access: "All repositories" or select your repos

4. In "Permissions", enable:

  • Pull requests → Read-only
  • Issues → Read-only
  • Metadata → Read-only (often enabled by default)

5. Click "Generate token" and paste it here.

Congratulations!
