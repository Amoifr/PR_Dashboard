How does scoring work?

Issue Scoring

Issues are scored based on maturity: +3 for "good first issue"/"help wanted" labels, +2 for maintainer comments, +1 for active discussion, +1 for detailed description, +1 for recent activity. Issues with 10+ comments get -1, old inactive issues get -2.

Filtering

Assigned issues, your own issues, issues with linked PRs, and issues without maintainer validation are automatically excluded.

Repository Discovery

Repos are auto-discovered from your merged PRs. You can also add repos manually. Results are cached locally until you refresh.

How to generate a token?

1. Go to GitHub Settings > Tokens

2. Click on "Generate new token"

3. Configure the token:

  • Token name: e.g.: "PR Dashboard"
  • Expiration: according to your preference
  • Repository access: "All repositories" or select your repos

4. In "Permissions", enable:

  • Pull requests → Read-only
  • Issues → Read-only
  • Metadata → Read-only

5. Click "Generate token" and paste it here.
